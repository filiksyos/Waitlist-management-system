Mebrej Speciality Dermatology Clinic - App Documentation

Application Overview

This is a patient management system for Mebrej Speciality Dermatology Clinic, designed to handle patient registration and queue management. The application consists of three main components:

  • Patient Registration Form (Receptionist.html)
  • Patient Queue Management (Doctor.html)
  • Public Display (Display.html)
